# Сканер СИЗ

Android приложение для сканирования средств индивидуальной защиты.

## Автоматическая сборка

APK автоматически собирается при каждом коммите в main ветку через GitHub Actions.

## Как получить APK

1. Перейдите во вкладку **Actions**
2. Выберите последнюю успешную сборку
3. В разделе **Artifacts** скачайте siz-scanner-apk

## Локальная разработка

```bash
python main.py
```

## Структура проекта

- main.py - основной файл приложения
- buildozer.spec - конфигурация сборки
- .github/workflows/android-build.yml - автоматическая сборка
